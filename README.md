# Controller

It uses SDL2 to get input from the joystick.


## C version


### Compiling

```bash
cd c
gcc main.c -o build/controller -lSDL2
```

### Running

```bash
cd c
./build/controller
```

## Rust version

### Compiling

```bash
cd rust
cargo build
```

### Running

```bash
cd rust
cargo run
```
